## ?? Quick Tips Visual

### **Image URL Best Practices**
```
? GOOD:
https://images.unsplash.com/photo-xxx?w=500&h=500&fit=crop
https://i.imgur.com/xxxxx.jpg
https://example.com/products/image.png

? BAD:
http://example.com/page (not direct image)
/local/file.jpg (relative path)
C:\Users\Desktop\image.jpg (local file)
```

### **Price Formatting**
```
? CORRECT:
$899.99
$45.00
$1,299.00

? INCORRECT:
899 (missing decimals)
$-50 (negative)
abc (not a number)
```
